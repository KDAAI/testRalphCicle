# Ralph Notes

Локальное Windows-приложение для заметок с тегами.

## Что умеет

- Создание, сохранение и удаление заметок.
- Поля заметки: заголовок, текст, теги.
- Теги вводятся через запятую.
- Поиск по заголовку, тексту и тегам.
- Один активный фильтр по тегу.
- Данные хранятся рядом с приложением в `notes.db`.

## Запуск из исходников

```powershell
python app.py
```

## Сборка одного exe

```powershell
.\build_exe.ps1
```

После сборки файл будет здесь:

```text
dist\RalphNotes.exe
```

При запуске собранного приложения база `notes.db` появится рядом с `RalphNotes.exe`.
