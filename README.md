# Ralph Notes

Локальное Windows-приложение для заметок с тегами, закреплением, автосохранением и JSON-резервными копиями.

## Возможности

- Создание, редактирование, сохранение и удаление заметок.
- Поля заметки: заголовок, текст, теги, дата создания, дата изменения и признак закрепления.
- Автосохранение существующих заметок через секунду после последнего изменения заголовка, текста или тегов.
- Новая заметка не попадает в базу автоматически: при уходе с заполненной новой заметки приложение предложит сохранить, отбросить или отменить действие.
- Закрепленные заметки показываются выше обычных и отмечаются префиксом `[Закреплено]`.
- Теги вводятся через запятую или точку с запятой, нормализуются и хранятся в отдельных таблицах SQLite.
- Можно выбрать несколько тегов одновременно; фильтр показывает только заметки, у которых есть все выбранные теги.
- Поиск работает по заголовку, тексту и тегам. Кнопка «Сброс» очищает поиск и выбранные теги.
- JSON-экспорт сохраняет все заметки, независимо от текущего поиска и фильтров.
- JSON-импорт добавляет заметки к текущей базе, пропускает дубликаты с теми же заголовком и текстом, принимает теги списком или строкой.
- Данные хранятся рядом с приложением в файле `notes.db`.

## Горячие клавиши

- `Ctrl+N` - новая заметка.
- `Ctrl+S` - сохранить текущую заметку.
- `Ctrl+I` - импортировать заметки из JSON.
- `Ctrl+E` - экспортировать все заметки в JSON.
- `Ctrl+F` - перейти в поиск.
- `Delete` - удалить выбранную заметку, если фокус не находится в поле редактирования.

## Запуск из исходников

Установите зависимости:

```powershell
pip install -r requirements.txt
```

Запустите приложение:

```powershell
python app.py
```

При запуске из исходников база `notes.db` создается в папке проекта.

## Сборка exe

Проект содержит существующий сценарий сборки Windows-исполняемого файла:

```powershell
.\build_exe.ps1
```

После успешной сборки файл будет здесь:

```text
dist\RalphNotes.exe
```

При запуске собранного приложения база `notes.db` создается рядом с `RalphNotes.exe`.

## Проверка

Основной набор тестов запускается так:

python -m unittest discover
